Requisitos para mandar stikers
- Formato: WEBP
- Tamaño: 512x512 píxeles (exacto, ni más ni menos)
- Peso máximo: 100 KB

Requisitos para mandar imagenes
- Formatos admitidos: .jpeg, .png (8‑bit, RGB o RGBA) 
- Tamaño máximo: 5 MB 

Requisitos para mandar PDFS
- Formatos de documentos permitidos: texto plano (.txt), PDF (.pdf), Word (.doc, .docx), Excel (.xls, .xlsx), PowerPoint (.ppt, .pptx) 
- Tamaño máximo: 100 MB
